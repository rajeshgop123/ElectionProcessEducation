# AI Election Process Education

A non-partisan AI-powered platform to educate users about the election process.

## Features
- ✅ AI chatbot for election rules & flow
- ✅ Verified, neutral explanations
- ✅ No political opinions or campaigning
- ✅ Easily extendable (quizzes, simulations)

## Tech Stack
- Frontend: React
- Backend: FastAPI
- AI: OpenAI API

## How to Run
### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
